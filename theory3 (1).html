Тип 3. Анализ логических высказываний. Теория

1. Основные логические операции

Логическая связка Логическая операция Другое название Истинно, если... Обозначение
НЕОтрицаниеИнверсияИсходное высказывание ложно¬
ИУмножениеКонъюнкцияОба условия выполняются одновременно∧
ИЛИСложениеДизъюнкцияВыполняется хотя бы одно из условий∨

2. Алгоритм решения

  1. Избавьтесь от «НЕ»: Раскройте скобки с отрицанием, перевернув знаки неравенств.
  2. Определите границы: Если стоит «И», вам нужно найти пересечение условий (число должно подходить под оба сразу). Если условий несколько, запишите их в ряд (например, 7 < X <= 15).
  3. Выберите число: Если нужно наименьшее — берите самое маленькое целое из подходящих. Если нужно наибольшее — берите самое большое.

3. Пример

Напишите наименьшее целое X, для которого истинно высказывание: НЕ(X <= 7) И (X чётное)

  1. Раскрываем НЕ(X ≤ 7) и получаем X > 7
  2. Условие теперь: (X > 7) И (X чётное)
  3. Числа, которые больше 7: 8, 9, 10...
  4. Наименьшее чётное из них — 8

4. На чём можно ошибиться:

  • Строгие и нестрогие знаки: Запомните: НЕ (>) — это ≤, а НЕ (≥) — это <. Это критично для выбора ответа.
  • Чётность: Ноль считается чётным числом.
  • Тип чисел: Внимательно читайте, нужно «целое число» или «натуральное» (натуральные начинаются с 1).
